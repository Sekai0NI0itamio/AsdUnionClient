# Minecraft 1.8.9 Client Modules List

This document provides a comprehensive list of all modules available in the Minecraft 1.8.9 client, organized by category with their corresponding functions.

## Client Modules

**Client modules** handle user interface, configuration, and client-side functionality.

- **Animations**: Controls entity animation settings and speeds
- **AutoConfig**: Automatically configures client settings
- **BrandSpoofer**: Spoofs the client brand to servers
- **ChatControl**: Manages chat-related settings and controls
- **ClickGUIModule**: Handles the click GUI interface
- **ConfigExport**: Exports client configurations
- **DefaultConfig**: Manages default configuration settings
- **FriendAdder**: Automatically adds friends
- **Friends**: Manages friend system functionality
- **GameDetector**: Detects specific game modes or servers
- **HUDModule**: Controls heads-up display elements
- **HudDesigner**: Allows customization of HUD layout
- **Rotations**: Manages player rotation settings
- **TargetModule**: Controls targeting system
- **Teams**: Manages team-related functionality
